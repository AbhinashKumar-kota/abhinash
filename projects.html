Active FTC - Mecanum Robot Paper in progress

Python ROS2 DMD Koopman Operator Raspberry Pi Arduino Mega

Data-driven fault estimation on a mecanum robot using DMD/Koopman - co-authored with IIT Mandi under ANRF grant CRG/2023/003333.

  • Applied Dynamic Mode Decomposition via Koopman operator to build a linear model directly from hardware data.
  • Luenberger observer estimates per-motor fault effectiveness in real time.
  • Fault effectiveness computed via direct closed-form algebraic derivation.
